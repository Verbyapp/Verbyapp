# Verby

<img src="./public/logo.png" alt="Verby" width="64" height="64" />

Master French verbs through play.

---

## Features

- **Blitz Mode** - 60 seconds. How many verbs can you conjugate correctly?
- **Ranked Duels** - Climb the global ladder in real-time PvP matches
- **Mistake Mastery** - Intelligent practice cycling back to your weak spots
- **Daily Quest** - 10 curated puzzles every day to maintain your streak

---

## Tech Stack

| Technology | Description |
|------------|-------------|
| [React](https://react.dev/) | UI library |
| [Vite](https://vitejs.dev/) | Build tool and dev server |
| [Tailwind CSS](https://tailwindcss.com/) | Utility-first CSS framework |
| [React Router](https://reactrouter.com/) | Client-side routing |

## Getting Started

```sh
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
